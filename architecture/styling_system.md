# SOP: Core Styling & Theme

## Goal
Establish a modern minimalist "Artist Portfolio" aesthetic.

## Design Tokens
- **Background:** Soft Beige (#FDFBF7)
- **Primary Text:** Charcoal (#4A4A4A)
- **Accent:** Gold (#D4AF37)
- **Font:** Serif for headings, Sans-serif for UI.
